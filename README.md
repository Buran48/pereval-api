# Pereval API

REST API для мобильного приложения "Перевалы" (Федерация Спортивного Туризма России)

## Описание проекта

API для приема и обработки данных о горных перевалах от туристов. Позволяет:
- Добавлять информацию о новых перевалах
- Редактировать данные (пока они на модерации)
- Просматривать статус модерации
- Получать список всех отправленных перевалов

## Задача от заказчика

ФСТР ведет базу горных перевалов, которая ранее пополнялась через PDF-отчеты по email.
Процесс занимал 3-6 месяцев. Новое мобильное приложение и API сокращают время обработки до 3 дней.

## Технологии

- **FastAPI** - веб-фреймворк для Python
- **PostgreSQL** - база данных
- **Pydantic** - валидация данных
- **Psycopg2** - драйвер PostgreSQL
- **Uvicorn** - ASGI-сервер
